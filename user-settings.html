When enabled, you'll need to enter a 6-digit code from the Google Authenticator app every time you log in. This protects your account even if your password is stolen.

How it works

  1. Download the free Google Authenticator app on your phone
  2. Click "Enable 2FA" below and scan the QR code
  3. Every login will require a 6-digit code from the app
  4. The code refreshes every 30 seconds for maximum security
